# 📚 Sistema de Gerenciamento de Biblioteca

Este projeto consiste em um sistema simples de gerenciamento de biblioteca desenvolvido em **Java**, utilizando os conceitos de **Programação Orientada a Objetos (POO)**.

O sistema permite realizar o cadastro de usuários e livros, além de controlar empréstimos e devoluções através de um **menu interativo no terminal**.

---

## 🎯 Objetivo do projeto

O objetivo deste projeto é aplicar conceitos fundamentais de programação, como:

- Programação Orientada a Objetos
- Encapsulamento
- Organização em pacotes
- Tratamento de exceções
- Manipulação de listas com `ArrayList`

---

## 🛠 Tecnologias utilizadas

- **Java**
- Estrutura de dados com **ArrayList**
- Conceitos de **POO**

---

## 📂 Estrutura do projeto

O projeto está organizado em diferentes pacotes para melhor organização do código:

- **model**  
  Contém as classes principais do sistema, como:
  - `Livro`
  - `Usuario`
  - `Aluno`
  - `Emprestimo`

- **service**  
  Contém a classe responsável pela lógica de funcionamento da biblioteca.

- **exception**  
  Contém a classe responsável pelo tratamento de exceções personalizadas do sistema.

- **enums**  
  Define o status dos livros (por exemplo: disponível ou emprestado).

- **principal**  
  Contém a classe principal responsável por executar o sistema.
